Interactive CLI tool to scaffold out a new [Gatsby](https://www.gatsbyjs.org/) post

## Why

* Generates folder structure and `frontmatter`
* Easy to use
* Interactive CLI

## Installation

```sh
npm install --global new-gatsby-post-cli
```
